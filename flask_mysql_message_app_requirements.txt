# Project Title

Flask MySQL Message Application

## Description

This project is a simple web application built using Flask and MySQL. It allows users to submit messages through a form and view stored messages on the web page. Messages are stored in a MySQL database.

## Features

- Submit and store messages in a MySQL database.
- View all stored messages on the website.
- Messages are persisted via a table named `messages` with an auto-incremented primary key.

## Requirements

- Python 3.x
- Flask (`>=2.0.0`)
- Flask-MySQLdb (`>=0.2.0`)
- MySQL Database

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```

2. Install dependencies:
   ```bash
   pip install flask flask-mysqldb
   ```

3. Configure environment variables to set up MySQL connection:
   ```bash
   export MYSQL_HOST=<your_mysql_host>
   export MYSQL_USER=<your_mysql_user>
   export MYSQL_PASSWORD=<your_mysql_password>
   export MYSQL_DB=<your_mysql_database>
   ```

4. Set up the MySQL database and table:
   - Use the provided `message.sql` file to create the required table:
     ```sql
     CREATE TABLE messages (
         id INT AUTO_INCREMENT PRIMARY KEY,
         message TEXT
     );
     ```

5. Run the application:
   ```bash
   python app.py
   ```

## Usage

1. Navigate to the homepage at `http://localhost:5000/`.
2. Add a new message using the form.
3. The message will be displayed on the homepage after submission.

## File Structure

- `app.py`: Main application file containing routes and logic for interacting with the database.
- `message.sql`: SQL script for setting up the `messages` table.

## Notes

- By default, the web server runs at `0.0.0.0:5000`.
- Make sure your MySQL server is running and accessible with the provided credentials.
